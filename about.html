Open Source Open Society is an event envisioned, designed and held collaboratively by 4 co-hosts:


Enspiral

Enspiral is a virtual and physical network of companies and professionals working together to create a thriving society.

GitHub

GitHub is the world’s largest host of open source code. Over eight million people use GitHub to build amazing things together.

Loomio

Loomio is free and open source software for anyone, anywhere, to participate in decisions that affect them.

Chalkle

Chalkle is a platform for open education designed to empower teachers and rewire the learning and teaching infrastructure in our communities.


Open Source Open Society is being delivered by a passionate team of collaborators from across the Enspiral network.
